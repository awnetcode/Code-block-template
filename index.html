Aby utworzyć blok z kodem należy użyć znaczników <pre> i <code> :

Ważne! Zamiast nawiasów ostrych < i > należy użyć encji &lt; i &gt;. Umożliwi to prawidłowe wyświetlenie całego kodu. W przeciwnym wypadku przeglądarka zinterpretuje zawartość <code> jako kod html. Aby nie robić tego ręcznie można użyć HTML escape (link).

HTML 
    
<div class="code-container">
    <div class="code-header">
        <span class="language-label">CSS</span>
        <button class="copy-btn" onclick="copyCode(this)">Skopiuj kod</button>
    </div>
<pre>
    <code id="code-block" class="language-css">
        /* Stylowanie dla &lt;pre&gt; */  
        pre {
            background-color: #2d2d2d;
            color: #f8f8f2;
            padding: 15px;
            border-radius: 8px;
            overflow-x: auto;       
        }
    </code>
</pre>
</div>

W kodzie CSS nie ma tego problemu chyba, że z jakiegoś powodu użyjemy nawiasów ostrych (na przykład w komentarzu).

CSS 
    
    /* Stylowanie dla <pre> */  
    pre {
        background-color: #2d2d2d;
        color: #f8f8f2;
        padding: 15px;
        border-radius: 8px;
        overflow-x: auto;       
    }

Do kolorowania składni można wykorzystać bibliotekę Prism. Aby to zrobić wystarczy umieścić w sekcji <head> poniższy kod:

Następnie należy dla <code> nadać odpowiednią klasę. Dla HTML będzie to .language-html

HTML 
    
<link href="https://cdnjs.cloudflare.com/ajax/libs/prism/1.28.0/themes/prism-tomorrow.min.css" rel="stylesheet" gt;
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.28.0/prism.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.28.0/components/prism-css.min.js"></script>

Aby dodać funkcję kopiowania kodu należy wykorzystać JavaScript:

W celu pokolorowania składni używamy klasy .language-js

HTML 
    
function copyCode(button) {
    const codeBlock = button.parentElement.nextElementSibling.querySelector('code');
    const code = codeBlock.innerText;

    // Skopiowanie kodu do schowka
    navigator.clipboard.writeText(code).then(() => {
        button.innerText = "Skopiowano!";
        setTimeout(() => {
            button.innerText = "Skopiuj kod";
        }, 2000);
    }).catch(err => {
        console.error('Nie udało się skopiować kodu: ', err);
    });
}
